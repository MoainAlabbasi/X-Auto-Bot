# 🚀 X-Auto-Bot: نظام الوكيل الذكي للتحكم بحساب X

هذا المستودع هو "محرك النشر التلقائي" الخاص بك على منصة X. تم تصميمه ليكون **Plug & Play**، مما يعني أنه يمكنك إعطاء رابط هذا المستودع مع "توكن قراءة" لأي وكيل ذكي (مثل Manus أو غيره) ليقوم بتنفيذ المهام نيابة عنك.

## 🛠️ المميزات:
- **نشر تلقائي:** نشر تغريدات فردية أو سلاسل (Threads).
- **جدولة ذكية:** إمكانية برمجة المهام للنشر في أوقات الذروة.
- **إدارة آمنة:** يتم سحب المفاتيح من **GitHub Secrets** لضمان أقصى درجات الأمان.

## 🔐 كيفية الإعداد (للمرة الأولى):
لكي يعمل هذا البوت، يجب إضافة مفاتيح X API الخاصة بك في إعدادات المستودع (GitHub Secrets) بالأسماء التالية:
1. `X_API_KEY`
2. `X_API_SECRET`
3. `X_ACCESS_TOKEN`
4. `X_ACCESS_TOKEN_SECRET`

## 🤖 تعليمات للوكلاء الأذكياء:
عندما يُطلب منك تشغيل هذا البوت:
1. قم بسحب (Clone) المستودع.
2. ثبت المكتبات المطلوبة من ملف `requirements.txt`.
3. تأكد من توفر مفاتيح البيئة (Environment Variables) المطلوبة.
4. قم بتشغيل `main.py` لتنفيذ المهمة المطلوبة.
